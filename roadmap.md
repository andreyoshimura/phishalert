# Roadmap

## Fase 1 - Fundacao

- Definir stack do backend
- Definir formato do evento
- Criar estrutura de API
- Criar armazenamento inicial

## Fase 2 - Captura

- Criar snippet para a página oficial
- Capturar `referrer`, IP, user-agent e timestamp
- Associar token de campanha

## Fase 3 - Correlacao

- Implementar normalização de sinais
- Correlacionar eventos com URLs suspeitas
- Marcar casos com alta confiança

## Fase 4 - Alerta

- Enviar alerta por canal configurável
- Gerar evidência técnica
- Registrar histórico do caso

## Fase 5 - Takedown

- Criar pacote de evidências
- Preparar notificação de abuso
- Automatizar fluxo de escalonamento

## Fase 6 - Operacao

- Dashboard de casos
- Regras de supressão e safelist
- Métricas de taxa de detecção e tempo de resposta
